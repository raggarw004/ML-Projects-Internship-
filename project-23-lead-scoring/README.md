# Project 23 — Lead Scoring Model (Business ML)

Lead scoring using logistic regression and calibrated probabilities.

## Approach
- Predict conversion probability from lead behavior features
- Calibrate probabilities for more realistic output
- Convert probabilities into business-friendly lead tiers

## Results
Lead score examples are saved in `reports/lead_scores.csv`.

## Tier Rules
- Hot: high conversion probability
- Warm: medium conversion probability
- Cold: low conversion probability

## Key Insight
Probability scores become more useful for business teams when they are translated into simple lead quality tiers.

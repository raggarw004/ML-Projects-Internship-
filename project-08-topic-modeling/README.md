# Project 08 — Topic Modeling (Unsupervised NLP)

This project applies topic modeling to discover latent themes in text data.

## Model
- Vectorization: TF-IDF
- Topic Model: NMF (4 topics)

## Key Insight
Topic modeling helps summarize large text collections without labeled data, enabling document exploration and content organization.

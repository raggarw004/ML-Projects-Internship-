# Project 20 — Anomaly Detection in Sensor Data

Anomaly detection on time-series sensor readings using Isolation Forest.

## Model
- Isolation Forest
- Unsupervised (no labeled anomalies)

## Results
Detected anomalies are shown as red points on the timeline.

## Interpretation
Most anomalies appear as sudden spikes outside the normal operating range, which could indicate sensor faults or abnormal system behavior.

## Key Insight
Unsupervised anomaly detection is essential when failure events are rare or poorly labeled.

# Project 10 — Resume Screening Model (Multi-class NLP)

This project classifies resumes into job categories using NLP.

## Model
- TF-IDF Vectorizer
- Logistic Regression (multi-class)

## Categories
- Software Engineer
- Data Scientist
- Product Manager
- Marketing

## ⚠️ Responsible Use & Ethics

This model is built **for educational purposes only**.

Limitations:
- Trained on synthetic data, not real hiring outcomes
- Text-only analysis ignores candidate context and experience depth
- May reflect biases present in training data

Responsible Use Guidelines:
- Do NOT use for automated hiring decisions
- Use only as a **supporting signal**, not a gatekeeper
- Combine with human review and structured interviews
- Regularly audit for bias and unintended discrimination

## Key Insight
AI systems used in hiring must prioritize fairness, transparency, and human oversight.

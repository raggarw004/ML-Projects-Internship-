# Project 24 — Customer Segmentation (Clustering)

Customer segmentation using RFM-style features and KMeans clustering.

## Approach
- Use recency, frequency, and monetary features
- Standardize features before clustering
- Evaluate cluster quality with silhouette score
- Interpret each cluster as a customer persona

## Results
Customer segment outputs are saved in `reports/customer_segments.csv`.

Cluster persona summaries are saved in `reports/cluster_personas.csv`.

## Key Insight
Clustering helps turn raw customer behavior data into useful business segments that can support targeting and retention strategies.

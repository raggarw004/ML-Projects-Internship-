# Project 22 — Product Recommendation (Content-based)

Content-based recommendation using product descriptions and TF-IDF.

## Approach
- Convert product text into TF-IDF vectors
- Compute cosine similarity between all products
- Return the most similar products for a selected item

## Results
A similarity demo is saved in `reports/similar_products.csv`.

## Key Insight
Products with similar text descriptions often represent similar use cases or customer needs.

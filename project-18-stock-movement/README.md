# Project 18 — Stock Movement Prediction

Binary classification to predict next-day stock movement using historical data.

## Methodology
- Feature engineering on returns and moving averages
- Walk-forward validation to avoid data leakage
- No random shuffling of time-series data

## Results
Accuracy is close to random, which reflects the difficulty of short-term market prediction.

## ⚠️ Disclaimer
This project is for educational purposes only.
It is NOT financial advice and should not be used for trading decisions.

## Key Insight
Most predictive power in markets is weak and unstable — robust evaluation matters more than headline accuracy.

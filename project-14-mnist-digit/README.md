# Project 14 — Handwritten Digit Classifier (MNIST)

Digit classification using a neural network trained on MNIST.

## Model
- Fully connected neural network
- Softmax output for digits 0–9

## Custom Inference
The model supports inference on user-provided handwritten digit images.

## Example Predictions
- Input: digit_3.png → Output: 3
- Input: digit_7.png → Output: 7

## Key Insight
Simple neural networks can generalize well when input preprocessing matches training data.

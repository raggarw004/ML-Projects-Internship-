# Project 17 — Energy Consumption Forecasting

Forecasting daily energy usage using time-series regression.

## Features
- Lag features (1, 7 days)
- Rolling averages
- Calendar features (weekday, month)

## Evaluation
Mean Absolute Error (MAE) used for evaluation.

## Error Analysis
The model shows higher error on weekends, indicating different consumption patterns compared to weekdays.

## Key Insight
Seasonality-aware features and error segmentation are critical for reliable energy forecasting systems.
